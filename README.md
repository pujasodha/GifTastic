# GifTastic

Game of Thrones themed gif selector. 
Gif's displayed when button is clicked, user has ability to add another button/character to search for gifs.
Gif's coming from Giphy.com

# Technologies / Frameworks Used 

HTML
CSS
Bootstrap
Jquery
API/ AJAX
